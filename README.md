# Hi there 👋

Welcome to my personal GitHub — this is where I keep my system configurations, scripts, and the tools I build to
make engineering, finance, and everyday workflows a little easier.

I'm a Mechanical Engineer based in Australia, with seven-plus years' experience in reliability, maintenance, and
capital projects across underground mining. Although my formal training is in mechanical engineering, mathematics,
and finance, I've always had a strong interest in programming and data science — hence most of what you'll find
here.

## What's in here
- **System & dotfiles** — A declarative, security-hardened NixOS configuration built around Emacs and the niri
  Wayland compositor.
- **Engineering tools** — Scripts and packages supporting CFD (OpenFOAM) and FEA (CalculiX/Code_Aster) workflows.
- **Finance & data** — Tools for tracking and analysing trading activity, built around hledger and a fair amount of
  Python.
- **General scripts** — Smaller utilities across Python, C++, Go, C, and Lua, mostly written to solve a specific
  problem I ran into.

## Using this code
I'm a firm believer in open source, and a lot of what's here exists because someone else's public repo saved me
time first. Feel free to use, fork, or adapt anything you find — if it's useful, it's yours.

something.)*
